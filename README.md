# AppRover backend

## Run ...

### Common
```shell
poetry shell

poetry install --only-root

poetry run {script_name} 
```

### Run start prod. server
```shell
poetry install --with dev

poetry run dev
```

### Run dev. server

```shell
poetry install --with dev

poetry run dev
```

### Run tests
```shell
poetry install --with dev
poetry run tests
```
